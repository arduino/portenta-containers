# Licenses
This project uses the licensed tool to manage the licenses of the dependencies. 
The licenses of the dependencies are stored in the `.licenses` directory. 
Each package has its own license file, which is generated by the licensed tool.

## How to update the dependencies licenses

- Install the licensed tool: https://github.com/licensee/licensed?tab=readme-ov-file#installation

- Install the taskfile tool: https://taskfile.dev/#/installation

- Run the task to update the licenses files:

```sh
    task license
```
